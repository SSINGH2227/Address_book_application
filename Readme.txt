Execution Steps:

1. Create a virtual environment
   cd C:\path\to\SID\Address_book_application
   pip install fastapi uvicorn

2. Run the FastAPI application:
   Run FastAPI application using Uvicorn.

3. Then, run the following command to start your FastAPI app:
   uvicorn Address_book_application:app --host 127.0.0.1 --port 4001

4. Open a web browser and navigate to http://127.0.0.1:4001
